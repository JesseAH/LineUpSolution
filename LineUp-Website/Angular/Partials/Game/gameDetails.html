How to create a Game?

Once you create a 'Game', any LineUp user can create a 'League' based on your game. As the game administrator your will have the following responsibilities

  • Create 'Sides' that will be matched up against each other.
  • Create 'Rounds'.
  • Create 'Matches' and mark the winners of those matches.
  • Respond to user feedback.
  • When the game is completed, mark it as finished and all of the winnings will be tallied up.
